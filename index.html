Dos formas: archivo simple (sin sync) o con Firebase ya incluido.
⚡ Recomendado: Exportar con Sync incluido
Genera un archivo .html con Firebase ya configurado. Tu socio lo abre y ya está sincronizado, sin tocar nada.
1
Descarga este archivo HTML (o usa "Exportar con Sync" de arriba)
En Claude.ai: botón de descarga ↓ bajo el mensaje
2
Envíalo por:
📱 WhatsApp: 📎 adjuntar → Documento
📧 Email: adjuntar y enviar normalmente
🍎 AirDrop: compartir archivo → AirDrop
☁️ Drive/iCloud: carpeta compartida
3
Tu socio lo abre en el navegador (Safari, Chrome…)
Sin instalar nada. Si WhatsApp lo abre en vista previa → "Abrir en navegador"
⚡ Sincronización en tiempo real
Configura Firebase para que los compañeros veáis el mismo estado al instante.
—
Cómo obtener los datos (5 min):
1. Ve a console.firebase.google.com
2. Crea un proyecto → Realtime Database → Crear base de datos
3. En Reglas pon: {"rules":{".read":true,".write":true}}
4. ⚙️ Configuración del proyecto → Tus apps → copia los datos
5. Todos los compañeros deben introducir los mismos datos
